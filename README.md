# sound-sharing

## Installation

```
npm run full-install
```

## Building your application

```
npm run build
```

## Starting webpack watch mode

```
npm start
```
### Starting dev server
```
live-server --port=3000 ./dist
```
## Running tests

```
npm test
```

## Accessing your application

http://localhost:3000/  

## Linting your code

```
npm run lint
```
